# 瑞小美 AiOS 项目专属配置

> 仅用于瑞小美 AiOS / 企微 AiOS 项目的额外配置

## 使用方式

在安装通用 harness 后，将本目录内容额外复制到项目的 `.cursor/` 目录中。

## 包含内容

### 专属规则
- **项目规范** — AiOS 系统的服务器、容器、数据库、前端技术栈约束
- **禁止硬编码** — 禁止在前端硬编码门户域名
- **小助手注册规范** — webhook 机器人的统一注册流程

### 专属命令
- **数据重置** — 清空老带新模块测试数据

### 业务技能（13个）
AI服务、企微对接、企微会话存档、聊天记录查看、员工选择器、客户筛选、钉钉对接、FastAPI基础设施、提示词工程、数据导出、文件上传、AI响应解析、实时通信
